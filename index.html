Git

Status, Pull, Branch

  1. git status - to check what branch we are working on
  2. git checkout main - switch to main branch
  3. git pull origin main - downloads the most up-to-date main branch
  4. git checkout -b feature/add-html - created a new branch with intent to add html features and switches to it

Commit, Confirm, Push

  1. git add -A - to commit all files in the working branch
  2. git commit -m "added code to index.HTML file"
  3. git pull origin main - downloads the most up-to-date main branch, it is good habit to always pull the base into your feature branch before pushing up and code
  4. git push origin feature/add-html - push all of our changes by using this command
